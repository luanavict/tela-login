#  Tela de Login

Projeto de uma tela de login responsiva e interativa desenvolvida com HTML, CSS e JavaScript.

---

## Sobre o projeto

Esta aplicação simula uma interface de login, com foco em design moderno e boas práticas de desenvolvimento front-end.

O objetivo foi praticar:
- Estruturação com HTML
- Estilização com CSS
- Manipulação de eventos com JavaScript

---

## Funcionalidades

-  Layout centralizado e responsivo  
-  Campo de email e senha  
-  Botão de login com validação  
-  Exibir/ocultar senha   
-  Validação de campos obrigatórios  
-  Efeitos visuais (hover e foco)

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## Layout

Interface com:
- Gradiente no fundo  
- Card centralizado  
- Inputs estilizados  
- Botão com efeito hover  

---

## Responsividade

O projeto se adapta a diferentes tamanhos de tela utilizando media queries.
